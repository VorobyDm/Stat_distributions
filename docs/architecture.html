Stat Distributions — Архитектура

Как устроено взаимодействие Frontend и Backend, и как работает анимация

1. Взаимодействие Frontend и Backend

Frontend (React + Vite :5173)

  • UI на Consta UI Kit
  • Графики на Plotly.js
  • Форма параметров (3 варианта)
  • Анимация (progressive reveal)
  • Статистики в сайдбаре
GET /api/distributions
список распределений + параметры
POST /api/compute
{distribution, variants, num_samples}
samples[] + theoretical + stats

Backend (FastAPI :8001)

  • scipy.stats — генерация выборок
  • scipy.stats — теоретические PDF/PMF
  • numpy — гистограммы, статистики
  • 11 функций распределений
  • JSON API, без состояния

2. Как работает анимация

Последовательность событий

FRONTEND Пользователь задаёт параметры 3 вариантов и нажимает «Сгенерировать»
FRONTEND fetch POST /api/compute — отправляет параметры на бэкенд
BACKEND scipy.stats.*.rvs() — генерирует все N сэмплов сразу для каждого варианта
BACKEND scipy.stats.*.pdf() — считает теоретическую кривую + np.histogram() + статистики
BACKEND Возвращает JSON: samples[1000], theoretical{x,y}, histogram{bins,counts}, stats{...}
FRONTEND Сохраняет полный ответ в results, ставит step = 0, запускает анимацию
ANIMATION setInterval(100ms) — каждые 100мс увеличивает step на speed (от 1 до 500)
ANIMATION samples.slice(0, step) — графики видят только первые step точек
ANIMATION SampleTraceChart: линии растут слева направо
HistogramChart: бары перестраиваются, приближаясь к теоретической кривой
FRONTEND step >= totalSamples — анимация останавливается. Пользователь видит итоговый результат

3. Формат данных (API)

POST /api/compute — Request:
{
  "distribution": "binomial",
  "variants": [
    { "n": 20, "p": 0.1 },
    { "n": 20, "p": 0.5 },
    { "n": 20, "p": 0.8 }
  ],
  "num_samples": 1000,
  "num_bins": 30
}
POST /api/compute — Response (per variant):
{
  "label": "Выборка 1",
  "color": "#e53935",
  "samples": [2, 1, 3, 0, ...], // N values
  "histogram": {
    "bin_edges": [-0.5, 0.5, 1.5, ...],
    "counts": [0.12, 0.28, ...] // density-normalized
  },
  "theoretical": {
    "x": [0, 1, 2, ..., 20],
    "y": [0.12, 0.27, ...] // PMF or PDF
  },
  "stats": {
    "mean": 2.01, "variance": 1.82,
    "std": 1.35, "median": 2.0,
    "min": 0.0, "max": 7.0
  }
}
